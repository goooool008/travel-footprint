# 🎨 旅行足迹 - 设计规范

> 版本：v1.0 | 最后更新：2026-06-24

---

## 一、设计理念

**关键词**：简洁 · 沉浸 · 星空 · 探索感

整体设计营造"在太空中俯瞰地球"的沉浸感。地球是视觉焦点，UI 元素尽量轻盈不遮挡视线。

## 二、色彩体系

### 主色调

| 用途 | 色值 | 说明 |
|------|------|------|
| 去过国家（亮色） | `#4FC3F7` 亮蓝 | 在地球上高亮显示 |
| 去过城市光点 | `#FFD54F` 暖金 | 温暖醒目 |
| 未去过区域（暗色） | `#1A237E` 深蓝黑 | 暗淡不抢眼 |
| 常驻城市标记 | `#FF7043` 珊瑚橙 | 特殊标记"家" |
| 海洋 | `#0D1B2A` 深海暗色 | 配合星空 |

| 用途 | 色值 |
|------|------|

## 三、星空背景

```
- 深色渐变底色：#000011 → #0A0A2E
- 星星：随机分布的白色小点，通过 CSS animation 实现微弱闪烁
- 可选装饰：微弱星云光晕（radial-gradient 叠加）
- 不遮挡地球，作为环境氛围
```

## 五、组件设计

### 5.1 搜索框
- 位置：左上角面板顶部
- 样式：半透明圆角输入框，placeholder 文字"搜索国家或城市..."
- 聚焦时边框发光（`#4FC3F7`）
- 下拉结果列表：匹配项高亮，点击添加

### 5.2 统计卡片
- 位置：右上角
- 4 个小卡片：国家数 / 城市数 / 覆盖面积 / 最远距离
- 卡片风格：毛玻璃效果，数字大而清晰

### 5.3 年份筛选器
- 位置：底部中央
- 样式：横向标签栏，每个年份一个可点击标签
- "一生" 标签始终在最左侧，默认选中
- 选中标签亮色，未选中灰色

### 5.4 国家信息面板
- 触发：点击国家进入国家视角时，从右侧滑入
- 内容：国旗 icon（emoji）、名称、首都、面积、人口、语言、货币、简介
- 带关闭按钮

### 5.5 按钮
- 圆角、轻微发光阴影
- hover 时变亮 + 微缩放
- 示例：导出按钮、导入按钮、设置"家"按钮

## 六、动效规范

| 场景 | 动效 | 时长 |
|------|------|------|
| 面板滑入/滑出 | translateX + opacity | 300ms ease |
| 统计数字更新 | 数字递增动画 | 500ms |
| 搜索结果出现 | 下拉展开 | 200ms |
| 国家高亮切换 | 颜色平滑过渡 | 500ms |
| 视角切换（全球↔国家） | globe pointOfView() 内置动画 | 1000ms |
| 星星闪烁 | CSS keyframes opacity 变化 | 2-4s 随机 |

## 八、响应式策略

- **桌面端（≥1200px）**：完整布局，侧边面板 + 统计卡片
- **平板端（768-1199px）**：面板可折叠，统计卡片缩小
- **手机端（<768px）**：不推荐使用（3D 地球在小屏幕体验差），但保留基本可用性
